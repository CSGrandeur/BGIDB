This is a demarcation demo of BGIDB, only the demarcation operation is applied. The whole
functions and source code of this project will be uploaded for public after this work is
published.

Operations：
Click 'Start', and then left mouse to pick key points, you can use undo, recover or reset to control
the demarcation.
When it is not started('Stop'), you can use middle mouse button to drag the image, and use mouse wheel
to zoom in or zoom out.
A gif is provided to explain how to operate.
